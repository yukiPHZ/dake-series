# DakePDFファイル名整理

PDFをドラッグ＆ドロップし、書類種別と相手名を指定するだけで、実務で探しやすいファイル名に整えるDAKEシリーズの単機能アプリです。

## アプリ概要

- PDFの中身は変更せず、ファイル名だけを変更します。
- 元PDFと同じフォルダ内でリネームします。
- 複数PDFをまとめて追加できます。
- PDF以外のファイルは受け付けません。

## 使い方

1. PDFを画面中央へドロップ、または「PDFを選択」から追加します。
2. 書類種別を選びます。
3. 相手名を入力します。
4. 「ファイル名を整える」を押します。
5. 完了後、最初のPDFがあるフォルダが開きます。

## ファイル名ルール

出力形式は次の通りです。

```text
書類種別_相手名様.pdf
```

例：

```text
売買契約書_山田様.pdf
売買契約書_山田様_2.pdf
売買契約書_山田様_3.pdf
```

相手名の末尾に「様」がない場合は自動で追加します。すでに「様」がある場合は重複させません。同名ファイルがある場合は連番を付け、既存ファイルを上書きしません。

## 注意事項

- PDF編集、OCR、PDF解析、書類種別の自動判定は行いません。
- 保存先の変更、履歴、細かな設定画面はありません。
- ファイル名に使用できない文字は全角文字へ置き換えます。
- PDFが開かれている場合、リネームできないことがあります。

## ビルド方法

```bat
build.bat
```

ビルドに成功すると、`dist\DakePDF_Rename.exe` が作成されます。共通アイコンは `..\..\02_assets\dake_icon.ico` を使用します。

## DAKEシリーズ共通思想

単機能、軽量、迷わない、現場で止まらない。多機能化せず、確実に「ファイル名を整える」ことだけに絞っています。
