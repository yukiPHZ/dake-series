# Dake固都税計算

DAKEシリーズの単機能アプリとして、土地評価額と建物評価額から固定資産税・都市計画税の概算年税額をすばやく確認するためのデスクトップツールです。入力変更ごとに即時計算し、起動から結果確認までを最短距離に絞っています。

## 概要

- 土地評価額と建物評価額を入力すると、固定資産税・都市計画税の概算を即表示します。
- 住宅用地区分の特例なし / 小規模住宅用地 / 一般住宅用地に対応しています。
- 都市計画税の ON / OFF、固定資産税率・都市計画税率の変更に対応しています。
- 参考計算に特化し、保存・印刷・PDF出力などの付加機能は入れていません。

## 使い方

1. `main.py` を実行します。
2. 土地評価額と建物評価額を入力します。カンマ付き入力も可能です。
3. 必要に応じて住宅用地区分、都市計画税の ON / OFF、税率を調整します。
4. 画面内に固定資産税、都市計画税、合計年税額、月額目安が即表示されます。

## 注意事項

- 本アプリは概算確認用です。複雑な固定資産税制度の完全再現は行いません。
- 税率は `%` 入力です。`1.4` は `1.4%` として扱われます。
- 未入力、文字入力、負数入力時は計算結果を表示せず、状態欄にやさしいエラーメッセージを表示します。
- 共通アイコン `..\..\02_assets\dake_icon.ico` を参照します。見つからない場合でもアプリは起動できます。

## 免責事項

入力値に基づく参考計算です。税務上の適正性を保証するものではありません。最終判断は税理士等の専門家に確認してください。

## ビルド方法

1. Windows 環境で `build.bat` を実行します。
2. PyInstaller により `dist/DakeFixedTax.exe` を生成します。

## DAKEシリーズについて

このアプリは DAKEシリーズの単機能アプリです。1機能 = 1アプリの考え方で、軽量・高速・安定・迷わない操作を優先しています。
