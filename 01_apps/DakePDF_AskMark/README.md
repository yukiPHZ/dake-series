# 聞くDAKE

PDF上の確認してほしい場所に「○」または「？」だけを付けて保存する、DAKEシリーズの簡易マークツールです。

## できること

- PDFを1つ開く
- 左クリックで「○」を付ける
- 右クリックで「？」を付ける
- Ctrl + Z または「1つ戻す」で直前のマークを消す
- 前後のページへ移動する
- Ctrl + マウスホイールで拡大・縮小する
- 元PDFとは別名で保存する

## 操作方法

1. 「PDFを選ぶ」またはドラッグ＆ドロップでPDFを開きます。
2. 確認してほしい場所を左クリックすると「○」が付きます。
3. 確認してほしい場所を右クリックすると「？」が付きます。
4. 必要に応じて「前へ」「次へ」でページを移動します。
5. 間違えたら Ctrl + Z または「1つ戻す」で直前のマークを消します。
6. 「保存する」で、元ファイル名に「_確認マーク」を付けたPDFとして保存します。

## 注意事項

- 元PDFは上書きしません
- ○と？以外の注釈機能はありません
- 確認依頼用の簡易マークツールです
- 正式な校正、契約判断、法的判断を代替するものではありません

## ビルド方法

事前に必要なライブラリをインストールします。

```bat
pip install pymupdf tkinterdnd2 pyinstaller
```

アプリフォルダで以下を実行します。

```bat
build.bat
```

`dist\DakePDF_AskMark.exe` が作成されます。
