# DakePDF Merge

DAKEシリーズ向けのPDF結合アプリです。追加したPDFを画面上で並べ替え、表示順どおりに1つのPDFへ結合します。

## ファイル構成

- `main.py`
- `requirements.txt`
- `build.bat`
- `README.md`

## アプリ情報

- 表示名: `PDF結合`
- exe名: `DakePDF_Merge.exe`
- 設定ファイル: `dake_pdf_merge_config.json`
- 共通アイコン: `..\..\02_assets\dake_icon.ico`

## 実行

```bat
python main.py
```

## ビルド

```bat
build.bat
```

配布対象は `dist\DakePDF_Merge.exe` です。
