# DAKE PDF Merge

既存の PDF 結合アプリをベースに、DAKE シリーズ向けの名前と構成へ整理したアプリです。

## ファイル構成

- `main.py`
- `requirements.txt`
- `build.bat`
- `README.md`
- `app.ico`
- `icon.png`

## アプリ情報

- 日本語アプリ名: `DakePDF結合`
- 英語表記: `DAKE PDF Merge`
- 設定ファイル: `dake_pdf_merge_config.json`
- 出力ファイル名末尾: `files_dake.pdf`

## 実行

```bat
python main.py
```

## ビルド

```bat
build.bat
```

## 依存関係

```bat
pip install -r requirements.txt
```
