# Dake工程カレンダー

## アプリ概要

指定した開始日から終了日までの期間をもとに、リフォーム工程表のようなカレンダー形式PDFを作成するDAKEシリーズ用アプリです。

ExcelやWordを使わず、現場名・期間・工程を入力してすぐにA4横PDFを出力できます。

## 使い方

1. `main.py` を起動します。
2. 現場名、支店名、担当者名、電話番号を入力します。
3. 開始日と終了日を入力します。
4. 各工程の「日付」と「内容」を入力します。
5. 保存先フォルダを選択します。
6. 「PDF作成」を押します。
7. 保存完了ダイアログのOK後、保存フォルダが開きます。

日付は `2025/11/29`、`2025-11-29`、`2025.11.29`、`20251129` の形式で入力できます。

## 入力項目

- 現場名
- 支店名
- 担当者名
- 電話番号
- 開始日
- 終了日
- 保存先
- 工程項目
  - 引渡し日
  - 水道設備の撤去
  - キッチン設置
  - ユニットバス設置
  - 大工工事
  - クロス工事
  - ハウスクリーニング
  - 完工

担当者名、電話番号、支店名、前回保存先は `dake_work_calendar_config.json` に保存され、次回起動時に再利用されます。

## PDF出力仕様

- 出力形式：PDF
- 用紙：A4横
- 日曜始まりのカレンダー
- 開始日〜終了日を含む連続週を表示
- 6週を超える場合は複数ページに分割
- 期間外の日付は薄く表示
- 月またぎ対応
- 日本の祝日表示
  - `holidays` ライブラリが利用できる場合は利用します。
  - 利用できない場合も、内蔵の簡易ルールで日本の祝日を表示します。
- 工程項目は該当日のセル内に表示
- 長い工程名は自動改行
- 文字量が多い場合はフォントサイズを小さくして収めます。
- 完工日が入力されている場合、下部に以下のコメントを自動表示します。

```text
完工は、{完工日}です。
いつもありがとうございます。引き続きよろしくお願いします！
```

下部には以下の形式で担当者情報を表示します。

```text
さくら都市{支店名} 担当：{担当者名} {電話番号}
```

## ビルド方法

同じフォルダ内で以下を実行します。

```bat
build.bat
```

`dist` フォルダに `DakeWork_Calendar.exe` が作成されます。

## 注意事項

このアプリは工程表作成を補助するためのものです。
日程・工程内容・完工日は、必ず利用者自身が確認してください。

初期実装では、ガントチャート化、工程の自動計算、業者別管理、案件データベース化、複数現場管理、クラウド保存、メール送信、印刷直接実行、ドラッグ操作による日程移動、Excel出力には対応していません。
